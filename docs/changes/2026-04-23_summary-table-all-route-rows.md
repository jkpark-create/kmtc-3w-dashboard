# 2026-04-23 월간 실적 표 전체 국가 행 표시

## 배경/문제

- Tab 1의 도착지/선적지 월간 실적 표에서 일부 국가가 보이지 않았다.
- 운영상 선적지별/도착지별 보기 모두 현재 필터 조건의 모든 국가 행을 확인해야 한다.

## 원인/판단

- `dist/index.html`의 `buildSummaryTable()`에서 행 목록을 WOS-3 BKG 기준 상위 15개로 제한하고 있었다.
- 기존 행 목록은 BKG 월간 집계에서만 만들었기 때문에 BKG에는 없고 BSA에만 있는 국가도 표 행에서 빠질 수 있었다.

## 결정/계산 로직

- 월간 실적 표의 상위 15개 제한을 제거한다.
- 행 목록은 현재 필터 조건의 BKG 국가와 BSA 국가를 합친 전체 목록으로 만든다.
- 정렬은 WOS-3 BKG 큰 순, BSA 큰 순, 국가 코드 알파벳 순으로 유지한다.

- `docs/changes/2026-04-23_summary-table-all-route-rows.md`

## 검증 결과

- 현재 `dist/data.json` 기준 도착지 BKG/BSA 합집합 33개 중 기존 상위 15개 제한으로 18개가 누락되던 것을 확인
- 현재 `dist/data.json` 기준 선적지 BKG/BSA 합집합 29개 중 기존 상위 15개 제한으로 14개가 누락되던 것을 확인
- 로컬 브라우저 검증에서 월간 실적 표가 도착지 33행, 선적지 29행으로 렌더링되는 것을 확인
