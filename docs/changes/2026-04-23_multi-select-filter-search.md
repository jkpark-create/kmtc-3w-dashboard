# 2026-04-23 멀티셀렉트 필터 검색

## 배경

- 선적지/선적포트/도착지/도착포트 멀티셀렉트 옵션이 많아지면서 원하는 국가나 포트를 찾기 어려웠다.
- 적용 버튼 방식은 유지하되, 메뉴 안에서 옵션 목록만 빠르게 좁히는 검색 기능이 필요했다.

## 변경

- 네 개 route 멀티셀렉트 메뉴 상단에 검색 입력을 추가했다.
- 검색어 입력 시 현재 메뉴의 옵션 목록만 필터링하고, 대시보드 cascade/render는 실행하지 않는다.
- 검색 결과가 없으면 `검색 결과 없음`/`No matches`를 표시한다.
- 메뉴를 닫거나 다시 열면 검색어를 초기화한다.
- 검색 input의 native `change` 이벤트가 필터 적용 이벤트로 오인되지 않도록, 멀티셀렉트 적용 이벤트에 `source: 'multi-select'` 마커를 추가했다.

## 검증

- `dist/index.html` 인라인 스크립트 문법 확인 완료.
- Playwright smoke test로 도착지 검색 `SG` 입력 시 옵션이 `SG`로 좁혀지고, 검색 중 render 0회, `적용` 후 render 1회를 확인했다.
