# 2026-04-23 월간 실적 표 합계 행 상단 고정

## 배경/문제

- Tab 1 월간 실적 표의 합계 행이 표 하단에 있어 전체 규모를 먼저 확인하기 어려웠다.
- 합계 행 순서가 `합계(주)` 다음 `합계(월)`로 표시되어 월 기준 합계를 먼저 보는 흐름과 맞지 않았다.

## 원인/판단

- `dist/index.html`의 `buildSummaryTable()`에서 국가별 행을 모두 렌더링한 뒤 마지막에 합계 행을 추가하고 있었다.
- 합계 행 라벨 배열도 주-월 순서로 구성되어 있었다.

## 결정/계산 로직

- 합계 행은 표 헤더 3줄 바로 아래에 렌더링한다.
- 월 전체 보기에서는 `합계(월)`을 먼저, `합계(주)`를 그 아래에 표시한다.
- 주차 선택 보기에서는 기존처럼 단일 `합계` 행만 표시한다.
- `합계(월)` 행은 월 기준 합계임을 구분하기 쉽도록 연한 노란색 배경으로 표시한다.
- 합계 계산 방식은 기존과 동일하게 유지한다.
