# 2026-04-23 대시보드 상단 영역 고정

## 배경

- 사용자가 대시보드를 아래로 스크롤해도 첨부한 상단 영역이 계속 보이도록 요청했다.
- 필터와 탭을 자주 바꿔야 하므로, 차트/테이블을 보다가 다시 상단으로 올라가지 않고 조작할 수 있어야 한다.

## 변경

- `dist/index.html`에서 헤더, 필터, 탭을 `.app-top` 컨테이너로 묶었다.
- `.app-top`에 `position: sticky; top: 0; z-index: 1000`을 적용했다.
- 멀티셀렉트 메뉴가 sticky 영역과 콘텐츠 위에 제대로 뜨도록 메뉴 z-index를 높였다.

## 검증

- `dist/index.html` 인라인 스크립트 문법 확인.
- Playwright smoke test로 900px 스크롤 후 `.app-top`의 y 좌표가 0으로 유지되는 것을 확인했다.
