# 2026-04-23 2025 BSA 백필 반영

## 배경

- 2025년 히스토리 데이터(`dashboard_summary_2025.json`)에 BSA가 0 rows로 들어가 있었다.
- 기존 BSA 다운로드 로직은 `vf_YYYY`/`vf_YYYYMM` URL 파라미터를 사용했지만, Tableau `BSArawBKGpattern` CSV export에서는 이 파라미터가 화면의 `YYYYMM` 필터를 바꾸지 못하고 기본 선택값을 유지했다.
- 사용자가 Tableau BSA 화면의 `YYYYMM` 필터에서 `202501`~`202512`를 선택해 다운로드해 반영해 달라고 요청했다.

## 원인/결정

- BSA 뷰는 CSV URL에서 visible filter caption인 `YYYY`, `YYYYMM`, `Sales Team`을 직접 넘겨야 필터가 반영된다.
- 테스트 결과 `vf_YYYYMM=202501,...,202512`는 2026 기본 월을 반환했지만, `YYYY=2025&YYYYMM=202501,...,202512`는 2025 전체 월을 반환했다.
- `daily_3w_dashboard.py`의 BSA 다운로드 파라미터를 `YYYY`/`YYYYMM` 방식으로 변경했다.
